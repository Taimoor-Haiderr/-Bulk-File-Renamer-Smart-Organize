# -Bulk-File-Renamer-Smart-Organize
A modern desktop tool built with Python and Tkinter that allows users to bulk rename files and organize folders intelligently using file type, date, or size — with preview, undo, and logging features.

#  Bulk File Renamer & Smart Organizer v2.0

A powerful and user-friendly desktop application for managing files efficiently. This tool allows you to **bulk rename files** and **organize folders automatically** using smart rules — all through a clean graphical interface.

# Features

#Bulk File Renaming

* Add **prefix / suffix**
* Find & replace text in filenames
* Auto numbering (custom start)
* Append current date (YYYYMMDD)
* Live **preview before applying changes**

#mart File Organization

Organize files into folders based on:

* File Type** (Images, PDFs, Videos, etc.)
* Date Modified** (e.g., `2026-04`)
* File Size** (Small, Medium, Large, Huge)

---

#Preview System

* See **before → after** changes
* Highlights modified files
* Prevents mistakes before execution


#Undo Support

* Undo last rename or organize action
* Safe and reversible operations

# Logging System

* Automatically logs all actions
* Export logs as:

  * CSV
  * JSON
* Includes timestamps and status tracking

#Drag & Drop Support

* Simply drag a folder into the app
* Automatically loads files


#Open Folder After Action

* Automatically opens the folder after renaming/organizing


#Tech Stack

* Python 3
* Tkinter (GUI)
* OS modules (os, shutil, pathlib)
* JSON & CSV for logging

#Usage

1. Drag & drop a folder or click **Browse**
2. Choose:

   * Rename options OR
   * Organize mode
3. Preview changes
4. Apply action
5. (Optional) Undo or save logs
